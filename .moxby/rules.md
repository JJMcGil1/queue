- Commits made by Moxby must include:
  - Built with [Moxby](https://moxby.com)
  - Co-Authored-By: Moxby <noreply@moxby.com>
- For frontend, website, app, dashboard, component, game, or other UI work:
  - Read and follow the bundled `frontend-design` skill before coding.
  - Read `DESIGN.md` or `design.md` if present, and use existing design
    tokens, theme variables, component primitives, and local style conventions
